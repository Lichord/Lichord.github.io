Anaconda基本使用

Anaconda来自于包管理和环境管理工具conda

Conda

  • 包管理与自带pip类似，管理Python第三方库
  • 环境管理能够允许用户使用不同版本Python,灵活切换
  • anaconda:一个集合，包括conda,某版本python,一批第三方库等

Spyder

IPython

  • ？
    • 在变量后面增加一个问号，获取变量信息
    • 函数后面增加一个问号，获取函数源代码
  • %run demo.py
    • 运行py程序，不在同一目录下需要输入绝对地址
  • % 魔术命令
