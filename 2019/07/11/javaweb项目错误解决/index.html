IDEA 导入别人的项目时报错：Invalid Maven home directory configured E:/apache-maven-3.5.2 Bundled maven 3.3.9 will be used

打开File setting找到maven修改maven 的配置地址就行

Error querying database. Cause: java.sql.SQLSyntaxErrorException: Unknown column ‘id’ in ‘field list’

  • 由于数据库表的列名多打了空格对应不上

  • 以后数据库的各种命名要做得更规范统一一些，不然很容易因为名字造成错误

Unable to load authentication plugin ‘caching_sha2_password’

数据库版本造成的问题,8.0.4之后修改了加密方式

https://blog.csdn.net/qq_22899021/article/details/83714816
