hexo+github Page搭建博客与配置

Jul 12, 2019 | github page

安装与配置Hexo

前提：安装了Node.js,Git，注册github账号，并且创建了一个{githubname}.github.io的仓库

  • 打开命令行输入 npm install -g hexo-cli
  • 本地新建空文件夹(文件夹名字可以随意取)
  • 目录切换到新建文件夹下
  • 初始化hexo init ./
  • 安装依赖 npm install
  • hexo generator
  • hexo s --debug
  • 在浏览器输入 http://localhost:4000 进入自己的博客

配置自己喜欢的主题

  • 我用的主题：https://github.com/A-limon/pacman
  • 切换到主题目录下cd themes/
  • clone到themes目录下git clone https://github.com/A-limon/pacman.git ./pacman
  • 修改根目录下的配置文件_config.yml,注意不是主题目录下的配置文件theme: pacman
  • 生成hexo g
  • 启动hexo s --debug
  • 在浏览器输入 http://localhost:4000 进入博客看到新主题

hexo部署到Github

  • 修改{githubname}.github.io目录下的配置文件_config.yml

  • 通过下面的命名进行博客静态页面的生成，以及部署到远端Github Pages
