3

shell中的特殊变量

  • $0 : shell或者shell脚本的名字
  • $* : 以一对双引号给出参数列表
  • $@ : 将各个参数分别加双引号返回
  • $# : 参数的个数
  • $_ : 上一个命令的最后一个参数
  • $$ ：所在命令的PID
  • $! : 最后执行的后台命令的PID
  • $? : 上一个命令执行后的退出状态 echo $? 返回值0，执行成功，否则失败

  • java实现多重继承： 一个类只能继承一个类，可以实现多个接口
  • 抽象方法不能有方法体

接口

interface中的属性在不提供修饰符的情况下会自动加上public static final

  • interface的属性不能改成private,protected,default
  • 接口中不能够定义方法体，所以无法通过get和set的方法获取属性值，所以属性值不属于对象，属于接口，默认使用static修饰
  • 接口中属性必须赋初值或者初始化，因为默认final修饰
